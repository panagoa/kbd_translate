ADVANCED VERBS ⭐⭐⭐ (chunks 031-044, pages 151-222, 492KB) CRITICAL
  ├─ TAM SYSTEM:
  │   ├─ Present tenses (Present 1, 2, 3)
  │   ├─ Past/Aorist (Perfect I -щ, Perfect II -рт, Aorist -гъа)
  │   └─ Future (Future I -ну, Future II -г(ъ)ащ)
  ├─ MOOD SYSTEM:
  │   ├─ Indicative (factual)
  │   ├─ Conditional (-н suffix, hypothetical)
  │   ├─ Conjunctive (-рэ, uncertain)
  │   ├─ Optative (-рэт, wish/desire)
  │   └─ Imperative (commands)
  ├─ NEGATION: -къым suffix (ALL tenses)
  ├─ DERIVATIONAL CATEGORIES:
  │   ├─ Causative (гъэ- "make/cause")
  │   ├─ Reciprocal (зэ- "mutual")
  │   ├─ Comitative (дэ- "with/together")
  │   ├─ Version (хуэ-/фIэ- "directional nuance")
  │   └─ Potential (хуэ- "ability")
  ├─ DIRECTIONAL PREVERBS:
  │   ├─ къэ- "away from"
  │   ├─ къы- "toward"
  │   ├─ щы- "upward"
  │   ├─ дэ- "downward"
  │   └─ те- "along surface"
  └─ Use: ESSENTIAL for kbd-translator (verb tense matching)

VERBAL DERIVATIONS (chunks 044-049, pages 216-247, 220KB)
  ├─ Participles (subject, object, adverbial)
  ├─ Gerunds (-у/-уэ, -рэ suffixes)
  ├─ Infinitives (-н, -ну, bare stem forms)
  └─ Use: Relative clauses, temporal expressions

ADVERBS (chunks 050-051, pages 246-256, 72KB)
  ├─ Temporal: джыри "now", пщэдей "yesterday", нобэ "today"
  ├─ Spatial: мыдрэ "here", ардрэ "there", дэнэ "where"
  ├─ Manner: -у suffix derivatives (дахэу "beautifully")
  └─ Use: Circumstantial expressions in translation

FUNCTION WORDS ⭐⭐ (chunks 052-055, pages 256-282, 152KB) CRITICAL
  ├─ POSTPOSITIONS (function like Russian prepositions):
  │   ├─ Locative: гъун "near", мыш "over", хьэ "below", фэ "in"
  │   ├─ Temporal: мыхьэнэ "after", дыдей "before"
  │   ├─ Causal: зэ "because of"
  │   └─ Comparative: хуэдэ "like"
  │   └─ All take possessive suffixes (-р, -м, -у)
  ├─ CONJUNCTIONS:
  │   ├─ Coordinating: и "and", ауэ "but", е "or"
  │   └─ Subordinating: фэ "that", зэрыIуэу "if/when", гъуэм "because"
  ├─ PARTICLES: гущэ "even", аркъудей "only", иIэ "let's"
  └─ Use: ESSENTIAL for sentence structure and connections

SYNTAX ⭐⭐⭐ (chunks 056-080, pages 282-404, 932KB) CRITICAL
  ├─ WORD ORDER: Subject-Object-Verb (SOV) - MANDATORY ⭐⭐⭐
  │   ├─ Rules in chunks 074-075 (pages 366-375)
  │   └─ Modifier position, case interaction
  ├─ COMPLEMENTS: Direct/indirect objects, oblique complements
  ├─ ATTRIBUTES: Prenominal modifiers, relative clauses
  ├─ COMPLEX SENTENCES: Coordination, subordination, embedding
  └─ Use: ESSENTIAL for kbd-translator (SVO→SOV transformation)

================================================================================
KEY TAKEAWAYS
================================================================================

1. GRAMMAR IS ORGANIZED LOGICALLY
   - Phonetics → Morphology → Syntax
   - Each section builds on previous

2. THREE CHUNKS ARE CRITICAL (read first!)
   - Chunks 074-075: SOV word order (absolutely essential)
   - Chunks 031-044: TAM/mood/negation (verb morphology)
   - Chunks 053-055: Postpositions/conjunctions (sentence structure)

3. USE GREP, NEVER USE READ ON LARGE FILES
   - All CSV files >3MB should use Grep with -m N limit
   - Never load entire 45MB corpus in memory

4. CORPUS-FIRST APPROACH FOR kbd-translator
   - Start with sents_292k.csv
   - Fall back to Rus-Ady_UASP.csv for unknown words
   - Use grammar for verification

5. SOV WORD ORDER IS MANDATORY
   - Russian: Subject-Verb-Object
   - Kabardian: Subject-Object-Verb
   - This is THE MOST IMPORTANT difference in translation
