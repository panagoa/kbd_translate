## Предложение 1: "Я вижу дом"

**Словарь:**
- видеть → лъагъун
- дом → унэ
- я → сэ

**Грамматика:**
- Я вижу → с- (1л.) + лъагъу = **слъагъу**
- дом (опред.) → **унэр** (-р)

**SOV:** Сэ (S) + унэр (O) + слъагъу (V)

**✅ ПЕРЕВОД: Сэ унэр слъагъу.**
