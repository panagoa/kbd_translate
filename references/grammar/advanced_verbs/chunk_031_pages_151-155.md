## Особенности трехличных глаголов с производными основами

### 4) Трехличный глагол с префиксом направления и каузатива

**Структура:** префикс прямого объекта + префикс направления + префикс косвенного объекта + префикс субъекта + префикс каузатива + корневая морфема

**Пример:** у-къ-е-з-гъэ-хь-а-щ «я заставил его нести тебя сюда»

### Особенности выражения косвенного объекта

В строении парадигмы личных форм глаголов с производными основами имеются некоторые особенности.

**Косвенный объект 3-го лица ед. числа** в глаголах, включающих префиксы версии или соучастия, выражен **нулевым аффиксом**:

- хуэ-с-хь-а-щ «я носил для него»
- НО: е-с-т-а-щ «я отдал ему»

### Противопоставление форм числа

Зато противопоставление форм числа 3-го лица косвенного объекта, за исключением форм, включающих префикс субъекта 3-го лица мн.числа, реализуется последовательно.

**Оппозитивные формы числа косвенного объекта 3-го лица:**

- дэ-с-хь-а-щ «я носил его (их) вместе с ним»
- я-дэ-с-хь-а-щ «я носил его (их) вместе с ними»

**Внеоппозитивные формы:**

- е-с-т-а-щ «я отдал ему (им)»
- е-п-т-а-щ «ты отдал ему (им)»

### Снятие противопоставления при субъекте 3 лица мн.числа

В производных глаголах (как и в простых) с префиксом субъекта 3-го лица мн.числа **а-** снимается противопоставление форм числа косвенного объекта 3-го лица:

- у-ху-а-пI-а-щ «они воспитали тебя для него (них)»
- у-д-а-ш-а-щ «они повели тебя вместе с ним (с ними)»

Существенные расхождения между простыми и производными глаголами обнаруживаются в оформлении огласовки личных аффиксов, что видно из материала таблиц парадигм личных форм (табл. 16).

## 8.6.7.3. Позиционная обусловленность полноты парадигмы

Степень дифференциации глагола по лицам косвенного объекта определяется не только строением основы и парадигматическим рядом.

### Позиционные ограничения в трехличных глаголах

Трехличные производные глаголы (с префиксом версии и соучастия) **позиционно ограничены** с точки зрения их дифференциации по лицам косвенного объекта.

Глаголы с указанными основообразующими префиксами не маркируют противопоставления форм числа косвенного объекта 3-го лица, если личная форма включает аффикс субъекта 3-го лица мн.числа.

#### Противопоставленные формы (субъект 1 л. ед.ч.)

- у-хуэ-с-ш-а-щ «я тебя повел для него»
- у-а-хуэ-с-ш-а-щ «я тебя повел для них»
- у-дэ-с-ш-а-щ «я тебя повел с ним»
- у-а-дэ-с-ш-а-щ «я тебя повел с ними»

В этих формах косвенный объект 3-го лица ед.числа выражается нулевым аффиксом, а косвенный объект 3-го лица мн. числа – префиксом **а-**.

#### Немаркированные формы (субъект 3 л. мн.ч.)

- у-ху-а-ш-а-щ «они тебя повели для него (них)»
- у-д-а-ш-а-щ «они тебя повели с ним (с ними)»

В этих формах лицо и число косвенного объекта выражены **нулем**.

### Характерные позиционные ограничения косвенного объекта 3-го лица

**Примеры:**

```
ху-а-пI-а-щ           «они воспитали его для него»
                      «они воспитали его для них»

у-ху-а-пI-а-щ         «они воспитали тебя для него»
                      «они воспитали тебя для них»

сы-ху-а-пI-а-щ        «они воспитали меня для него»
                      «они воспитали меня для них»
```

### Морфологическая омонимия

Ущербность парадигмы трехличных переходных глаголов показывает формы типа:

```
ху-а-пI-а-щ           «они воспитали его для него»
                      «они воспитали его для них»
                      «они воспитали их для него»
                      «они воспитали их для них»
```

Отсутствие противопоставления форм числа прямого и отчасти косвенного объектов 3-го лица создает **широкую базу для морфологической омонимии** в языке.

## 8.6.8. Четырехличные и пятиличные переходные глаголы

Ниже даются парадигмы четырехличного глагола (табл. 17).

### Общая характеристика

Четырехличные и пятиличные глаголы **могут быть только производными**. Они образуются с помощью префиксов каузатива, версии и соучастия.

**Пример образования четырехличного глагола:**

Посредством каузации трехличный глагол **хэшы-н** «выводить» становится четырехличным:

гупым у-а-х-е-з-гъэ-ш-ащ «я заставил его вывести тебя из группы (людей)»

**Пример образования пятиличного глагола:**

Осложнение последнего префиксом соучастия или версии ведет к преобразованию его в пятиличный глагол:

гупым у-а-ды-х-е-з-гъэ-ш-а-щ «я заставил его вывести тебя с ним из группы (людей)»

### Последовательность морфем

Морфемы четырехличных и пятиличных переходных глаголов имеют следующую последовательность:

#### 1) Четырехличный глагол с префиксом соучастия (или версии)

**Структура:** префикс прямого объекта + префикс косвенного объекта + префикс соучастия (или префикс версии) + префикс косвенного объекта + префикс субъекта + корневая морфема

**Пример:** у-а-д-е-с-шэ-лI-а-щ «я тебя вместе с ним подвел к нему»

#### 2) Четырехличный глагол с префиксами направления, локализации и каузатива

**Структура:** префикс объекта + префикс косвенного объекта + префикс направления + префикс локализации + префикс косвенного объекта + префикс субъекта + префикс каузатива + корневая морфема

**Пример:** гупым д-а-къы-х-е-б-гъэ-ш-а-щ «(из группы людей) ты заставил его (велел ему) вывести нас»

#### 3) Пятиличный глагол с префиксами направления, версии (или соучастия) и каузатива

**Структура:** префикс прямого объекта + префикс косвенного объекта + префикс версии (или соучастия) + префикс косвенного объекта + префикс субъекта + префикс каузатива + корневая морфема

**Пример:** у-а-ху-ре-з-гъэ-шэ-лI-а-щ «я заставил его (велел ему) подвести тебя к нему для них»

### Ограничения в пятиличных глаголах

В пятиличном глаголе могут быть выражены положительными аффиксами, как правило, **не более двух косвенных объектов** (третий косвенный объект выражается нулевым аффиксом).

### Распространенность и продуктивность

Формы четырехличных глаголов, вопреки распространенному мнению, **часто встречаются в языке**.

**Продуктивные формы четырехличных глаголов:**

- схуритащ «он дал ему что-то для меня»
- схужыриIащ «он сказал ему что-то для меня»

**Продуктивные формы пятиличных глаголов:**

- къысхухебгъэшащ «ты заставил его вывести что-то из чего-то для меня»
- хутырезгъэлъхьащ «я заставил его класть что-то на что-то для него»

### Ограничения полноты парадигмы

Однако **не все** четырехличные (не говоря уже о пятиличных) глаголы образуют полную парадигму личных форм. Полнота парадигмы зависит от строения личной формы.

**Грамматически правильная форма:**

- схуригъэшащ «он заставил его вести что-то для меня» (версия) — грамматически правильная и стилистически нейтральная

**Невозможная форма:**

- *соотносительная форма соучастия в значении «он заставил его вести что-то вместе со мной» — невозможна

**Возможные формы соучастия от трехличного глагола егъэшэ-н:**

- дезгъэшащ «я заставил его вести что-то вместе с ним»
- удезгъэшащ «я заставил его вести тебя вместе с ним»
- сыдебгъэшащ «ты заставил его вести меня вместе с ним»

### Структура аффиксов

Структура аффиксов лица, версии и соучастия (их фонетическое оформление, ступень огласовки) четырехличных и пятиличных глаголов имеет некоторые [продолжение на следующей странице]
